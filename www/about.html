About The Project

WebServ is a comprehensive HTTP server implementation written in C++ 98, designed to handle multiple clients simultaneously without blocking.

Our Mission

To understand the intricacies of the HTTP protocol and network programming by building a server from scratch. We aim to provide a lightweight, compliant, and educational web server.

Technology Stack

  • • C++ 98 Standard
  • • Non-blocking I/O (poll/epoll/kqueue)
  • • HTTP/1.1 Protocol
  • • CGI (Common Gateway Interface)
